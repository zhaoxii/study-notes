
<!DOCTYPE> 文档类型声明，作用就是告诉浏览器使用哪种html版本来显示网页

<!DOCTYPE html>  这句话的意思是，这个页面使用html5版本来显示网页

<!DOCTYPE> 声明位于文档中的最前面的位置，处于<html>之前

<!DOCTYPE> 不是html标签，只是文档类型声明标签

<html lang="en">  lang是语言种类。用来定义当前文档显示的语言。

<meta charset="UTF-8">  规定html文档使用哪种字符编码。 必须要写，不写的话会出现乱码


p  段落标签.

段落之间有空隙

br  换行 ,单标签

标题标签，h1-h6

strong , b  加粗.  推荐strong,语义更强烈
em,i  倾斜,推荐em,语义更强烈
del,s 删除线,推荐del,语义更强烈
ins,u 下划线,推荐ins,语义更强烈



div,span 标签,没有语义,他们就是一个盒子,用来装内容.

div 用来布局,单独占一行. 大盒子
span 一行可以放多个.   小盒子


img  图像标签,单标签
<img src="url"/>   src是img标签的必须属性,用于指定图像文件的路径和文件名.
alt属性
title属性

border属性  ,图像边框

a 标签 ,超链接标签
<a href="" target=""></a>
href 跳转目标 . 必须属性
target 目标窗口的弹出方式. 默认_self当前页面打开. __blank在新窗口打开

外部链接。必须加http/https。如https//www.baidu.com
内部链接：网站内部页面之间的相互链接，直接链接内部页面名称即可。如：herf="index.html"
空链接： herf="#"
下载链接：herf里是文件地址或者压缩包时，则会下载

锚点链接：点击之后，快速定位到网页中的某个位置。
href中，设置属性值为#名字的形式，如href="#two"。同时，在目标标签定义一个id="two"的属性

表格标签
主要用于展示数据，因为可以使数据展示的非常的规整，可读性较好。
<table></table>  定义表格
<tr></tr>   定义行
<td></td>   定义单元格

<table>
	<tr>
		<td></td>
	</tr>
</table>

<th></th>    表头单元格
表头单元格也是单元格，常用于表格第一行。其中的文字加粗居中显示。

表格属性，不常用，通常是通过css来设置。(table标签的属性)
align   left，right，center    规定表格相对周围元素的对齐方式
border   1或""           是否有边框。默认是""，没有边框 
cellpadding     像素值        规定单元边沿与其内容之间的空白。默认1像素
cellspacing     像素值        规定单元格之间的空白。默认2像素
width           像素值或百分比       表格的宽度



表格结构标签 (放在table标签内)
表格分割成 表格头部和表格主体。 主要是用来划分结构，使结构清晰
<thead>   表格的头部区域 。 一般是包含第一行。
<tbody>   表格的主体区域。  不包含第一行


合并单元格
跨行合并  rowspan="合并单元格的个数"。 写在最上侧单元格
跨列合并  colspan="合并单元格的个数"。 写在最左侧单元格

合并步骤：
1、确定跨行还是跨列
2、找到目标单元格，写rowspan 或 colspan
3、删除多余的单元格

列表标签
表格是用来显示数据的，那么列表是用来布局的。
列表的特点就是整齐，整洁，有序，作为布局会更加自由和方便。

无序列表，有序列表，自定义列表

无序列表(重点)
ul 标签。
<ul>
	<li>列表项</li>
	<li></li>
	<li></li>
</ul>
没顺序。
ul中只能嵌套li，直接在ul中放其他标签或者文字是不允许的。
li相当于一个容器，可以放其他元素。
无序列表会带有自己的样式属性，实际中，通常用css来设置。


有序列表
ol 标签定义有序列表，使用li来定义列表项。
<ol>
	<li>列表项</li>
	<li></li>
	<li></li>o
</ol>
有顺序，第一个li是排第一，后面依次。
ol中只能嵌套li，直接在ol中放其他标签或者文字是不允许的。
li相当于一个容器，可以放其他元素。
有序列表会带有自己的样式属性，实际中，通常用css来设置。


自定义列表(重点)
dl 标签用于定义描述列表(自定义列表)，该标签与dt，dd一起使用
dt定义项目/名字，dd描述每一个项目/名字
<dl>
	<dt>名词1</dt>
	<dd>名词1解释1</dd>
	<dd>名词1解释2</dd>
</dl>

<dl>
	<dt>关注我们</dt>
	<dd>官方微信</dd>
	<dd>新浪微博</dd>
	<dd>联系我们</dd>
</dl>

dl中只能包含dt，dd。
dt，dd个数没限制，通常是一个dt，对应多个dd
dt，dd中可以放任何标签。

表单标签

在html中，一个完整的表单通常由表单域，表单控件(也叫表单元素)和提示信息3部分组成。


表单域
表单域是包含表单元素的区域。

form标签用于定义表单域，以实现信息的传递和收集。
form会把他范围内的表单元素信息提交给服务器。

<form action="url地址" method="提交方式" name="表单域名称">
	各种元素控件
</form>

action  指定接收并处理表单数据的服务器程序的url地址。
method  指定提交方式，get/post
name    指定表单的名称，以区分同一个页面中的多个表单域


表单控件(表单元素)

----------------
input 输入标签
<input type="属性值"> 
根据type属性的不同，输入字段拥有多种形式。(文本字段，复选框，单选按钮等)。下面是type可取的一些值。
text     文本框
submit   提交按钮，点击后会把表单数据发送到服务器。
reset    重置按钮，清除表单中的所有数据 
radio     单选按钮，同一组定义同一个name属性。
password   密码字段，该字段中的字符被掩码。
image		定义图像形式的提交按钮
hidden		定义隐藏的输入字段
file       上传文件
checkbox   复选框。同一组定义同一个name属性。
button     定义可点击按钮，啥也不做，多数情况下，可通过js执行脚本。


name属性   定义input元素的名称。每个input元素的name应该是不一样的
value属性  规定input元素的值。(初始值)
checked     规定此input首次加载时，应该被选中
maxlength    规定输入字段中的字符的最大长度

单选按钮和复选框可以设置checked属性，当页面打开的时候就选中某个值

-----
label标签
该标签为input元素定义标注
label标签用于绑定一个表单元素，当点击label标签内的文本时，浏览器就会自动将光标转到对应的表单元素上，增加用户体验。

label的for 属性与相关元素的id属性值一样。
<label for="username2">用户名：</label><input type="text" id="username2">


-----------
select 
下拉表单元素

在页面中，如果有多个选项，并且想要节约页面空间，可以使用select

<select>
	<option>选项1</option>
	<option>选项2</option>
	<option>选项3</option>
</select>

select 中至少包含一对option
在option标签中定义selected="selected"时，即为默认选项




-----------
textarea
文本域元素

使用场景，当用户输入内容较多时。
在表单元素中，textarea 标签是用于定义多行文本输入的控件。

<textarea cols="30" rows="10">
	文本内容(当打开时，默认显示的内容)
</textarea>

rows  显示的行数
cols  每行显示的字符数
这两个属性实际不会用，实际开发中使用css进行开发。

视频  video

使用它们可以很方便的在页面中嵌入音频和视频，而不再去使用flash和其他浏览器插件。

video支持的视频格式：mp4，webm，ogg。
尽量使用mp4格式，可以在所有浏览器中播放。

<video src="视频地址" controls="controls"></video>


autoplay  autoplay	如果出现该属性，则视频在就绪后马上播放。
controls	controls	如果出现该属性，则向用户显示控件，比如播放按钮。
height	pixels	设置视频播放器的高度。
loop	loop	如果出现该属性，则当媒介文件完成播放后再次开始播放。
muted	muted	如果出现该属性，视频的音频输出为静音。
poster	URL	规定视频正在下载时显示的图像，直到用户点击播放按钮。
preload	auto/metadata/none	如果出现该属性，则视频在页面加载时进行加载，并预备播放。如果使用 "autoplay"，则忽略该属性。
src	URL	要播放的视频的 URL。
width	pixels	设置视频播放器的宽度。
